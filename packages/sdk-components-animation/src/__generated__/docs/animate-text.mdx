# Text Animation

Text animation allows you to split text by char or by word to animate it.

## Installation

Use the Webstudio component registry item `animate-text`.

In Builder automation, resolve the item from the generated shadcn-compatible registry and use the Webstudio metadata in `meta` for insertion.

Inspect the registry metadata before creating or editing instances so content model, states, and composition stay valid.

## API Reference

### Props

- `className`: type: `string`, control: `text`. Classes to which the element belongs
- `easing`: type: `string`, control: `select`, default: `"linear"`. Easing function applied within the sliding window.
- `slidingWindow`: type: `number`, control: `number`, default: `5`. Size of the sliding window for the animation:
- 0: Typewriter effect (no animation).
- (0..1]: Animates one part of the text at a time.
- (1..n]: Animates multiple parts of the text within the sliding window.
- `splitBy`: type: `string`, control: `select`, default: `"char"`. Defines how the text is split for animation (e.g., by character, space, or symbol).
