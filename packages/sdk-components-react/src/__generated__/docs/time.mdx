## API Reference

### Props

- `country`: type: `string`, control: `select`, default: `"GB"`.
- `dateStyle`: type: `string`, control: `select`, default: `"medium"`.
- `datetime`: type: `string`, control: `text`.
- `format`: type: `string`, control: `text`. Custom format template. Overrides Date Style and Time Style.

Tokens: YYYY/YY (year), MMMM/MMM/MM/M (month), DDDD/DDD/DD/D (day), HH/H (hours), mm/m (minutes), ss/s (seconds)

Examples:

- "YYYY-MM-DD" → 2025-11-03
- "DDDD, MMMM D" → Monday, November 3
- "DDD, D. MMM YYYY" → Mon, 3. Nov 2025

Day and month names use the selected language.

- `language`: type: `string`, control: `select`, default: `"en"`.
- `timeStyle`: type: `string`, control: `select`, default: `"none"`.
- `timeZone`: type: `string`, control: `timeZone`, default: `"UTC"`. Timezone used to display the date. Use "visitor" to display each visitor’s browser timezone after the page loads, or select/type an IANA timezone like "Europe/Berlin".
