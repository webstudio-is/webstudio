# Image

Add an image asset to the page. Webstudio automatically converts images to WebP or AVIF format and makes them responsive for best performance.

## Installation

Use the Webstudio component registry item `image`.

In Builder automation, resolve the item from the generated shadcn-compatible registry and use the Webstudio metadata in `meta` for insertion.
