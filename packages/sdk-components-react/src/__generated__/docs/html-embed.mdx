# HTML Embed

Used to add HTML code to the page, such as an SVG or script.

## Installation

Use the Webstudio component registry item `html-embed`.

In Builder automation, resolve the item from the generated shadcn-compatible registry and use the Webstudio metadata in `meta` for insertion.

## Usage

Use this component through Webstudio's instance, props, styles, resources, and data-source model.

Inspect the registry metadata before creating or editing instances so content model, states, and composition stay valid.

## Composition

This component is used as a standalone component or follows its content model.

## API Reference

### Props

- `className`: type: `string`, control: `text`.
- `clientOnly`: type: `boolean`, control: `boolean`. Activate it for any scripts that can mutate the DOM or introduce interactivity. This only affects the published site.
- `code` required: type: `string`, control: `code`.
- `executeScriptOnCanvas`: type: `boolean`, control: `boolean`. Dangerously allow script execution on canvas without switching to preview mode. This only affects build mode, but may result in unwanted side effects inside builder!

- Direct children: `ws:descendant`
