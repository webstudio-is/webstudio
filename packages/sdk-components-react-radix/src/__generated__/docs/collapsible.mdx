# Collapsible

Collapsible Webstudio component.

## Installation

Use the Webstudio component registry item `collapsible`.

In Builder automation, resolve the item from the generated shadcn-compatible registry and use the Webstudio metadata in `meta` for insertion.

Prefer the `Collapsible` template when creating a new instance from scratch.

## Usage

Use this component through Webstudio's instance, props, styles, resources, and data-source model.

Inspect the registry metadata before creating or editing instances so content model, states, and composition stay valid.

## Composition

Use the following composition to build `Collapsible`:

```text
Collapsible
|-- CollapsibleTrigger
`-- CollapsibleContent
```

## Examples

Use the `Collapsible` template as the canonical starter. An interactive component which expands and collapses some content, triggered by a button.

## API Reference

### Props

- `disabled`: type: `boolean`, control: `boolean`. Whether the form control is disabled
- `open`: type: `boolean`, control: `boolean`. Show or hide the content of this component on the canvas. This will not affect the initial state of the component.

### Content Model

- Category: `instance`
- Direct children: `instance`

- Closed: `[data-state="closed"]`
