# Radio group

Radio group Webstudio component.

## Installation

Use the Webstudio component registry item `radio-group`.

In Builder automation, resolve the item from the generated shadcn-compatible registry and use the Webstudio metadata in `meta` for insertion.

Prefer the `RadioGroup` template when creating a new instance from scratch.

## Usage

Use this component through Webstudio's instance, props, styles, resources, and data-source model.

Inspect the registry metadata before creating or editing instances so content model, states, and composition stay valid.

## Composition

Use the following composition to build `RadioGroup`:

```text
RadioGroup
`-- RadioGroupItem
    `-- RadioGroupIndicator
```

## Examples

Use the `RadioGroup` template as the canonical starter. A set of checkable buttons—known as radio buttons—where no more than one of the buttons can be checked at a time.
