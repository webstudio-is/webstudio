# Popover

Popover Webstudio component.

## Installation

Use the Webstudio component registry item `popover`.

In Builder automation, resolve the item from the generated shadcn-compatible registry and use the Webstudio metadata in `meta` for insertion.

Prefer the `Popover` template when creating a new instance from scratch.

## Usage

Use this component through Webstudio's instance, props, styles, resources, and data-source model.

Inspect the registry metadata before creating or editing instances so content model, states, and composition stay valid.

## Composition

Use the following composition to build `Popover`:

```text
Popover
|-- PopoverTrigger
`-- PopoverContent
    `-- PopoverClose
```

## Examples

Use the `Popover` template as the canonical starter. Displays rich content in a portal, triggered by a button.
